## 📚 Acerca del proyecto

**AeroSafe Quality** es una empresa especializada en consultoría y asesoría para industrias farmacéutica, cosmética, veterinaria y alimentaria, farmacias de manipulación y hospitales. Ofrece servicios de cualificación de equipos, validación de procesos y seguridad biológica, garantizando el cumplimiento normativo y la excelencia operativa.

## 🛠️ Stack tecnológico

### Frontend
- [Next.js](https://nextjs.org) — Framework React para aplicaciones web
- [React](https://reactjs.org) — Biblioteca para interfaces
- [TailwindCSS](https://tailwindcss.com) — Estilos responsivos
- [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) — Animaciones al hacer *scroll*
- [ShadCN](https://ui.shadcn.com) — Componentes de UI accesibles

### Infraestructura
- [Firebase Hosting](https://firebase.google.com/docs/hosting) — Alojamiento

## 📋 Mapa del sitio

- **Inicio** — Panorama de la empresa y de los servicios
- **Sobre nosotros** — Misión, visión y valores
- **Servicios** — Detalle de servicios (tratamiento de aire, salas limpias, seguridad biológica, entre otros)
- **Equipos** — Equipos usados en cualificación y validación de procesos

## 💼 Servicios

- **Tratamiento de aire** — Mantenimiento y control de sistemas de climatización
- **Salas limpias y ambientes controlados** — Certificación y cualificación de áreas críticas
- **Cualificación de equipos** — IQ, OQ y PQ para el cumplimiento normativo
- **Validación de procesos** — Seguridad biológica y calidad en entornos industriales
- **Análisis de gases industriales** — Cualificación de aire comprimido y otros gases
