## 📚 About the project

**Sixess - English School** is an innovative English school that combines blended learning and active methodologies to accelerate language learning. Founded by **Leo Oliveira**, the school offers courses for children, teens, and adults, using an interactive digital platform to enhance the learning experience.

## 🛠️ Tech stack

### Website
- [HTML5](https://html.spec.whatwg.org/) and [CSS3](https://www.w3.org/Style/CSS/) — Structure and styling
- [JavaScript](https://developer.mozilla.org/docs/Web/JavaScript) — In-browser behavior

### Infrastructure and tools
- [Firebase Hosting](https://firebase.google.com/docs/hosting) — Hosting
- [Flexge](https://www.flexge.com) — Interactive learning platform
- [Zoom](https://zoom.us) and [Google Meet](https://meet.google.com) — Online classes

## 📋 Site map

- **Courses** — Options for different ages and levels
- **Methodology** — Blended learning, flipped classroom, and a communicative approach
- **Ranking** — Gamification with discounts for top students
- **Schedule** — Weekly class and event planning
- **Pricing and services** — Plans, duration, and fees
- **Contact** — Enrollment and student support

## ✨ What sets it apart

- **Placement tests** for better course fit
- **Personalized lessons** based on student performance
- **Gamification and interactive activities** to drive engagement
- **Certification recognized under the CEFR** (Common European Framework of Reference for Languages) upon course completion
